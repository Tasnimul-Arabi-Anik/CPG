# Study Pilot

## Current objective

Audit the current Klebsiella phage project data and produce one real pilot analysis report using established tools and reviewed source data where available.

## Pilot deliverables

- `results/pilot/data_audit.tsv`
- `results/pilot/tool_run_summary.tsv`
- `results/pilot/rbp_candidate_summary.tsv`
- `results/pilot/host_range_summary.tsv`
- `results/pilot/hypothesis_feasibility.tsv`
- `PILOT_REPORT.md`

## Stop condition

The pilot is complete when the report states concrete usable data counts, inspected tool/source outputs, commands and versions where available, and which biological claims remain blocked.

## Scope control

Do not add new workflow frameworks, dashboards, placeholder schemas, custom biological predictors, or platform abstractions for the pilot. Use existing tools directly and write only minimal glue for summaries and joins.
